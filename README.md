## Project Overview
This project is a Power BI dashboard created to analyze Uber ride data and visualize key ride patterns, demand trends, and customer behavior.

## Key Analysis Areas
- Total bookings and cancellations
- Peak demand hours
- Location-wise ride trends
- Revenue and payment patterns
- Customer ride behavior

## Tool Used
- Power BI

## Dashboard Highlights
- Interactive KPIs for total rides and cancellations
- Time-based analysis (hour/day-wise)
- Location-wise ride distribution
- Clean and user-friendly dashboard design

## Conclusion
This dashboard demonstrates the ability to analyze ride data and present insights clearly using Power BI for business decision-making.
